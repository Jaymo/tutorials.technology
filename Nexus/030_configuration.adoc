== Configuration of Nexus
	
Once the nexus repository manager has been started, its web interface can be accessed under this URL:

....
http://localhost:8081/nexus/
....
	
The Nexus contains some repositories by default:

=== User Settings

The default user is
`admin`
and the password is
`admin123`.
