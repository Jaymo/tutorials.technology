== Support for the Git version control systems
	
Jenkins supports the Git version control system via a plugin.
Select the menu:Manage Jenkins[Manager Plugins] link. 
Here you have to install the _Git Plugin_.
	
To clone a Git repostory via Jenkins you need to enter the email and user name for your Jenkins system. 
For this switch into your job directory and run the `git config` command.
