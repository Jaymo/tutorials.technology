[[shortcuts]]
== Shortcuts

[[shortcuts_windows]]
=== Windows

* kbd:[WindowsKey + d] - Hide / restore all applications
* kbd:[WindowsKey + m] - Minimize all applications
* kbd:[WindowsKey + r] - Open the run dialog
* kbd:[WindowsKey + Pause] - Opens the settings dialog
