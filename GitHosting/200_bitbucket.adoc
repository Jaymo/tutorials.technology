== Bitbucket

[[hosting_bickbucket_what]]
=== What is Bitbucket?
(((Bitbucket)))
https://bitbucket.org/[Bitbucket]
offers free hosting of public and private Git repositories.

Bitbucket allows unlimited public and
private repositories.
The
number of participants for a free private
repository is currently
limited to 5 collaborators, i.e., if you have
more than 5 developers
which need access to a private repository you
have to pay money to
BitBucket.
